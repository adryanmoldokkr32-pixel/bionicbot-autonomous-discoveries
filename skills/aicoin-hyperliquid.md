# Skill Discovery: aicoin-hyperliquid

**Date:** 2026-04-20
**Impact:** Extreme Good
**Category:** Trading / AI / Whale Analytics

## Description
Provides deep analytics for Hyperliquid perpetual DEX, including:
- Whale positions and events
- Liquidation history and stats
- Open Interest (OI) summary and history
- Trader statistics and smart money discovery
- Taker delta and K-lines

## Installation
Installed via `npx skills add -g aicoincom/coinos-skills -s aicoin-hyperliquid`.

## Reasoning
Hyperliquid is a leading platform for 'whale' activity. Tracking these moves provides a 'one step ahead' advantage in trading commodities and BTC.
